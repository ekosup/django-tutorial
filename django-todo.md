# Django Tutorial

---

## Installing Django
To install django we can type in the command line:
```
pip3 install django
```

---
## Starting a project

```
python3 manage.py startproject theprojectname
```

---
## Run development server
to check if the project is successfully created

```
python3 manage.py runserver
```
